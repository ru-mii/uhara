## Syntax
```c#
IntPtr CreateTool(
[in] string grand
[in] string sub
);
```   
## Parameters
**[in] grand**   
Broad tool name.   
<br>
**[in] sub**   
Sub category of the tool.
## Return value
Returns object with type of created tool.
Returns null if failed or not found.   
## Remarks
Has to be used to use tool functions.
